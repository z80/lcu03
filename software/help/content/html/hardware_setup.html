Hardware setup dialog

This setup dialog has several functions.
  • Adjust laser power control driver
  • Adjust polarization control driver
  • Choose current shutter position
  • Find current drivers position
  • Upgrade device firmware if new firmware's been released

Laser power driver adjustment

It is necessary to select Power in driver selection combo box in the very top of the dialog. After that one may adjust driver position in two ways. 1) To set position in appropriate box explicitly or 2) use incremental left and right buttons with step increment input area.

Polarization driver adjustment

It is necessary to select Polarization in driver selection combo box in the very top of the dialog. After that in the same way one may adjust driver position by either setting it explicitly or by using use incremental left and right buttons.

Shutter position

In order to choose current shutter position one may select it via "Shutter opened" and "Shutter closed" radio buttons.

Finding current drivers position

After any service work it is a good practive to tel the system find it's dirver mechanical positions. In order to do that one may use "Find pos." button.

WARNING! This operation will perform full driver revolution. So in order to prevent sensitive sample from being damaged it is recommended to perform this operation with shutter closed or laser off.

Firmware upgrade

In order to use this function one is to have a new firmware file for the device. Having the file one may upgrade internal device firmware by clicking this button and choosing the file in open file dialog.

WARNING! Do not unpower the device during firmware upgrade operation. It may cause it's bricking. Unbricking the device requires using another software utility which is not a part of software installation package.
