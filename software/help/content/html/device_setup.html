Device setup

Driver installation

In order to install driver for the hardware one should browse to software installation folder and locate driver subfolder there. In driver subfolder there are two executable files for 32-bit and 64-bit computer architectures. One should execute aprpopriate file for the computer.

Device selection

In order to start working with the hardware one need to setup the hardware properly in the software. One is to go to "device" falling menu and select virtual COM port which corresponds to the hardware.
